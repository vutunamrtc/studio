# **App Name**: PaisaPal

## Core Features:

- Transaction Input: Allow users to input their income and expenses with descriptions, amounts, and categories.
- Category Management: Enable users to create, edit, and delete transaction categories to better organize their finances.
- Income/Expense Tracking: Track income and expenses over time, allowing users to visualize their spending habits.
- Spending Summary: Generate spending summaries by category to help users identify areas where they can save money.
- Budgeting Tool: A tool where the LLM uses information about the user's historical spending to advise them about setting monthly budgets.

## Style Guidelines:

- Primary color: A muted green (#77DD77) to evoke a sense of financial health and growth.
- Background color: A light, desaturated green (#F0FFF0).
- Accent color: A light orange (#E6B800) to draw attention to important actions and key metrics.
- Body and headline font: 'PT Sans', a humanist sans-serif, will provide a modern look while maintaining a touch of warmth.
- Use simple, clean icons to represent different transaction categories. The style should be modern and easily recognizable.
- Design a clean and intuitive layout that makes it easy for users to input transactions, view summaries, and manage categories. Use clear visual hierarchy to guide the user's attention.
- Incorporate subtle animations for feedback on user interactions, such as adding a transaction or navigating between screens. The animations should be smooth and unobtrusive.
